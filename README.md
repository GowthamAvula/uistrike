# UIStrike - Flight Navi

A premium flight booking application built with Next.js, featuring stunning UI animations and a seamless user experience.

## Features

- **Interactive Landing Page** with animated 3D globe and flight routes
- **Flight Search & Booking** with real-time results
- **Flight Customization** - Choose seats, meals, and add-ons
- **Traveler Details & Payment** with form validation
- **Digital Boarding Pass** with holographic effects
- **Premium Animations** using Framer Motion
- **Responsive Design** optimized for all devices

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS + Mantine UI
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Language**: TypeScript

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
